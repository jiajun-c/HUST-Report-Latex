# 华中科技大学报告Latex模板

![](https://img.shields.io/badge/linux-archlinux-blue)
![](https://img.shields.io/badge/build-pass-green)

## 1. 目标
打造让huster专注内容的Latex自动化模板
可以根据输入的课程和学生信息自动生成模板，huster只需要填写实验内容即可。

## 2. 使用

推荐vscode + Latex插件，随后直接编译即可(linux)，没有windows电脑不太确定(
